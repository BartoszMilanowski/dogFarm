[PL] Projekt stanowiący stronę internetową przykładowej hodowli psów rasowych. W pracy wykorzystałem HTML, CSS oraz JavaScript. Strona jest responsywna. Nieaktywne linki wynikają z faktu prowadzenia do podstron przedstawiąjących widoki zaprezentowane na innych podstronach, tylko ze zmienioną treścią. Projekt można zobaczyć pod adresem: https://dogfarm.netlify.app/

[EN] A project consisting of a website for an exemplary purebred dog breeding. I used HTML, CSS, and JavaScript in this work. The website is responsive. Inactive links lead to subpages showcasing views presented on other subpages, only with different content. You can view the project at: https://dogfarm.netlify.app/
